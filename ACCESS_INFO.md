### 🔧 トラブルシューティング

#### アクセスできない場合
1. **ファイアウォールの確認**
   ```bash
   sudo ufw status
   ```
   必要に応じてポートを開放：
   ```bash
   sudo ufw allow 5001
   sudo ufw allow 5174
   ```

2. **サービスの稼働確認**
   ```bash
   ss -tuln | grep -E ":(5001|5174)"
   ```

3. **ネットワーク接続確認**
   ```bash
   ping 192.168.3.135
   ```
